# macsiem.github.io

Root GitHub User Pages site serving `macsiem.dev`. Hosts `app-ads.txt` at root for AdMob (publisher pub-5353750048775501) and a small landing page that points to:
- mobile app store links + privacy policies (Mote, Compresser, Moje Szczepienia, Archvr, DeMotion)
- public Home Assistant tooling repos (ha-* cards and integrations on GitHub).

Files:
- `app-ads.txt` — AdMob ads.txt declaration; required at the eTLD root for AdMob app verification (Mote, Compresser, Moje Szczepienia, Archvr, DeMotion all use the same publisher)
- `index.html` — landing page (mobile apps + HA Tools sections)
