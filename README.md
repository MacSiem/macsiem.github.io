# macsiem.github.io

Root GitHub User Pages site. Serves `app-ads.txt` at root for AdMob (publisher pub-5353750048775501) and a small landing page that points to the app privacy policies hosted in per-app repos (e.g. `mote-privacy`).

Files:
- `app-ads.txt` — AdMob ads.txt declaration; required at the eTLD root for AdMob app verification (Mote, Compresser, Moje Szczepienia, Archvr, DeMotion all use the same publisher)
- `index.html` — minimal landing
